# Sound provenance

All WAV files in this directory are original procedural assets generated by
`quadrant_wars/ui/sound.py`. They are synthesized from mathematical waveforms,
envelopes and deterministic noise; no third-party samples or game audio are
bundled.

Combat variants cover footsteps, sword swings, metal and shield impacts, stone
hits, Soldier deaths, reinforcements and objective clashes. Runtime channel
limits and per-event cooldowns keep large battles readable.
